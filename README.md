# AI Student Assistant

The **AI Student Assistant** is a FastAPI-based application that leverages Hugging Face models to generate summaries, multiple-choice questions (MCQs), and exam questions from lecture notes or text. This project aims to help students generate useful study materials and exam preparation resources automatically.

## Features

- **Summarization**: Generate concise summaries of long lecture notes or documents.
- **Multiple-Choice Questions (MCQs)**: Automatically generate MCQs based on provided content.
- **Exam Questions**: Create exam-style questions to help students with their studies.
- **Health Check Endpoint**: Simple health check route to ensure the API is running smoothly.

## Technologies Used

- **FastAPI**: Web framework for building APIs with Python.
- **Hugging Face API**: Used for natural language processing (NLP) tasks like summarization, question generation, etc.
- **Uvicorn**: ASGI server to run the FastAPI app.
- **Python 3.x**: Programming language used for the backend.
